DBMS for Retail Analytics

Overview
A retail data management system built in T-SQL, focused on integrity and business analytics.

What this repository includes:
* Schema: Table structure with PK/FK constraints.
* Initial data: Data sets for testing relationships.
* Analytics: Dynamic views for sales and inventory reports.

 Upcoming:
* Stored procedures for order automation.
* Audit triggers for price security.
* Performance optimizations through indexing.
